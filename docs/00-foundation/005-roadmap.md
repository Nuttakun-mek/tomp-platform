# Product Roadmap

## Phase 1: Foundation

Goal: Establish the working kernel.

Scope:

- Authentication
- Organization
- User
- Project
- Operation Day
- Session
- Mission
- Assignment
- Call Sign
- Driver
- Vehicle
- Timeline

## Phase 2: Operation Visibility

Goal: Support live operations.

Scope:

- Driver QR access
- Driver activation
- Vehicle photo and plate photo
- GPS updates
- Mission Control
- Status updates
- Notifications
- Google Maps links

## Phase 3: Coordination

Goal: Reduce calls and improve coordination.

Scope:

- Coordinator workspace
- Organizer viewer
- Change requests
- Incident handling
- Recovery workflow
- Contact matrix

## Phase 4: Knowledge and Reporting

Goal: Capture and reuse operational knowledge.

Scope:

- Closing review
- Lessons learned
- Venue notes
- Vendor performance
- Driver performance
- Reports
- Export

## Phase 5: Intelligence

Goal: Add rule-based intelligence before AI.

Scope:

- Risk scoring
- Readiness scoring
- SLA tracking
- Decision suggestions
- Capacity planning

## Phase 6: Platform Scale

Goal: Support larger operations and partners.

Scope:

- Vendor portal
- Customer portal
- Public API
- Advanced permissions
- Advanced analytics
- Optional paid integrations

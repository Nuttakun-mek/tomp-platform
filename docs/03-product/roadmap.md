# Roadmap

## Phase 0: Foundation

- Establish repository structure and documentation canon.
- Define core domain language.
- Set up Next.js, TypeScript, Tailwind CSS, Supabase, and Vercel foundations.
- Define database migration conventions.
- Define ADR process.

## Phase 1: Planning Baseline

- Projects.
- Missions.
- Drivers.
- Vehicles.
- Call signs.
- Assignments.
- Draft plan editing.
- Plan publication.

## Phase 2: Field Access and Navigation

- QR-based driver access.
- Driver mobile assignment view.
- Google Maps deep links.
- Basic driver status updates.
- PWA installability.

## Phase 3: Live Operations

- Operations dashboard.
- Coordinator-scoped views.
- Organizer-scoped views.
- GPS tracking visibility.
- Timeline entries.
- Change event recording.

## Phase 4: Review and Improve

- Operation replay timeline.
- Exception review.
- Change event summaries.
- Post-operation reporting.
- Exportable operational records.

## Phase 5: Scale and Hardening

- Advanced access policies.
- Performance tuning.
- Offline resilience.
- Realtime improvements.
- Operational analytics.

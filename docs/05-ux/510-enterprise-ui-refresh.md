# TOMP Enterprise UI Refresh

วันที่อัปเดต: 02/07/2026 00:27 น. Asia/Bangkok

## เป้าหมาย

ยกระดับหน้าระบบหลักจากรูปแบบ prototype ให้เป็นระบบปฏิบัติการองค์กรที่อ่านง่าย สบายตา และเหมาะกับการทดสอบภายใน โดยยังคงขอบเขตเดิมของ TOMP คือการวางแผน จัดสรร ติดตาม GPS ดู Timeline และจัดการรายการที่ต้องติดตาม

## แนวทางภาพรวม

- ใช้พื้นหลังสีอ่อนแบบ enterprise workspace
- ใช้ sidebar สีเข้มเพื่อให้รู้สึกเป็น command platform
- ใช้ panel ที่มีระยะห่างและ hierarchy ชัดเจน
- ใช้สีสถานะเท่าที่จำเป็น: เขียวสำหรับพร้อม, น้ำเงินสำหรับเส้นทาง/GPS, เหลืองสำหรับต้องติดตาม, แดงสำหรับเสี่ยง
- ใช้ภาษาไทยเป็นหลัก และคง English เฉพาะ TOMP, Call Sign, QR, GPS, Google Maps, Mission Control

## หน้าที่ปรับ

- App Shell และ Navigation
- Dashboard / ภาพรวมการปฏิบัติการ
- Mission Control / ศูนย์ควบคุมปฏิบัติการ
- Driver mobile card
- Resource overview
- Project readiness summary

## ข้อจำกัด

รอบนี้เป็นงาน UI/UX refresh ไม่ได้เพิ่ม scope business ใหม่ และไม่ได้เปลี่ยน database schema

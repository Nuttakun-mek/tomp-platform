# UX Blueprint v1.0

## UX Principles

1. Driver mobile-first.
2. Operation desktop-first.
3. Coordinator mobile and tablet friendly.
4. Organizer simple viewer/change requester.
5. Dark mode from the beginning.
6. Responsive layouts for mobile, tablet, desktop, and wide screen.
7. Every notification must have action.
8. Every major screen must answer an operational question.
9. Avoid unnecessary popups.
10. Use cards, timelines, boards, and action panels.

## Information Architecture
